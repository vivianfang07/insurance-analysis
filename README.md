# Insurance Expense Analysis

Analyzing the relationship between insurance expenses and key factors 
such as age, BMI, and smoking status using the Kaggle insurance dataset.

## Dataset
[Insurance CSV](https://www.kaggle.com/datasets/awaiskaggler/insurance-csv)

## Methods
- Exploratory Data Analysis (EDA)
- One-Hot Encoding
- Linear Regression (R² = 0.865)

## Key Findings
- Smoking status is the strongest predictor of insurance expenses (r = 0.787)
- Age has a moderate positive correlation with expenses (r = 0.299)
- Adding a smoker × BMI interaction term improved R² from 0.783 to 0.865
